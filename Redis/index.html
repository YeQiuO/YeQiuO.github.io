Redis 基础

先速通一遍，底层实现后续慢慢看书学

Redis 的特性：单线程、原子性、内存、数据结构、I/O 多路复用的非关系型数据库

6.0 版本前网络 I/O 和命令处理都是单线程，6.0 版本后网络 I/O 采用多个线程处理

  • 大部分操作都在内存中完成。CPU 并不是制约 Redis 性能表现的瓶颈所在，而是机器的内存或者网络带宽
  • 避免了多线程之间的竞争
  • I/O 多路复用机制
